# sm-entDB
#### 아이돌 그룹의 상세정보와 음반 그리고 각 멤버의 프로필 사진 및 상세정보를 테이블 형태로 출력하는 홈페이지
- 웹을 기반으로 SM엔터테이먼트 소속 아이돌 그룹의 DB를 설계
- 환경: 웹 환경에서 Next.js 및 React 사용, Groom IDE 사용
